# sismo-maps
Ionic project with custom map layers.

## Description
Main diffrences from the Official SismoAlarm App is;
  * Plate boundry data with 2 diffrent types
  * Another layer for tectonic plates  
  * Bottom modal to open and close map layers (Better UX for mobile usage)
  * Layer info screens inside the modal

## Setup
1. Clone the repository.
2. Install dependencies:
```bash
npm install
```
3. Run development server or build as needed.
